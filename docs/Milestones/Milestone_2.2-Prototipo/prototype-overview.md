# Prototype Overview

## Current State of the Prototype

Our current implementation includes:

- Core gameplay functionality;
- Three working gamemodes;
- Multiple autonomous agents with different behaviours.

The prototype validates the main concepts proposed for the project.

---

# Agent Design

The application currently supports three different types of virtual agents.

## Random Agent

Plays using almost entirely random decisions.

- Serves as a baseline opponent;
- Makes no strategic considerations.

## Weak Agent

Simulates an inexperienced player.

- Plays without deep strategic planning;


## Sharp Agent

Provides a more challenging experience.

It uses heuristics that consider:
- Card strength;
- Trick value;
- Partner status;
- Basic Sueca strategy.

---

# Gamemodes

The application currently supports three different ways of playing Sueca.

## Virtual Mode

Play entirely within the mobile application using virtual players.

## Physical Mode

Traditional gameplay supported by the application.

The app assists players by managing the game and reducing manual effort.

## Hybrid Mode

Combines physical and digital gameplay.

Some participants play traditionally, while others participate virtually through the application.
