# Project Overview

The **Sueca Application with Virtual Player** is a mobile application designed to modernise the traditional Sueca card game by combining the physical and digital worlds.

The application allows players to:
- Play Sueca entirely within the app;
- Use autonomous virtual players;
- Mix real and virtual players in the same match;
- Simplify game management and organisation.

## Stakeholders

- Tournament organisers
- IPSS
- Companies interested in recreational activities

## Users

- Players of Sueca
