## Expected Functionalities (MVP)

The Minimum Viable Product (MVP) focuses on core functionality that makes in-person Sueca games easier to run and fairer to play. The main expected features:

- **Virtual referee**: A lightweight referee component that coordinates the flow of the game, enforces turn order and detects obvious rule violations to help keep games running smoothly.
- **Score calculation**: Automatic tracking and calculation of points and rounds so players don't need to manually tally scores.
- **Simple game-state interface**: A minimal, clear UI showing the current game state (players, scores, current turn, recent plays) so everyone can follow along.
- **Card recognition**: Use the smartphone camera to recognise cards on the table and identify which cards were played each turn.
- **Cheating identification**: Detection heuristics to flag inconsistency or potential cheating (e.g., impossible card combinations or mismatched plays) so a human referee can review and resolve disputes.

These features were prioritised to deliver a functioning, useful prototype that demonstrates the viability of computer-vision-assisted game management.
