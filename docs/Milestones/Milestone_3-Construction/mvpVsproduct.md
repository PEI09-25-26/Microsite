## MVP vs Product

This page contrasts the Minimum Viable Product (MVP) scope with a full product vision.

**MVP (Prototype)**

- Focus: Prove the core idea quickly and reliably.
- Deliverables: Working virtual referee, card recognition pipeline, automatic score tracking, and a simple UI demonstrating game-state updates.
- Constraints: Limited features, narrower device support, reduced UX polish; built to validate the concept and collect feedback.

**Product (Future vision)**

- Focus: Production-ready, robust, performant and delightful user experience.
- Deliverables: Polished UX, cross-device support, full tournament management, cloud sync, analytics dashboards, multi-language support, accessibility improvements, and continuous AI model updates.
