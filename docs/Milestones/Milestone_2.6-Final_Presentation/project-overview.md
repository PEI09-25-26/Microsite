# Project Overview

A Mobile app for the card game “Sueca” that allows real world play with an autonomous virtual agent or a regular virtual player, mixing the real world with the virtual world.

---

## Problems

### Manual Counting
Frequent errors in Points Calculation

### No Referee
It’s hard to pay attention if someone cheats

### Missing Players
Impossible to play Sueca with less than 4 players

There was no adequate digital solution that would fix all these problems of a Traditional Game of Sueca

---

## Solution

- Mobile Application  
- Card Recognition  
- AI Players  
- Automatic Referee
