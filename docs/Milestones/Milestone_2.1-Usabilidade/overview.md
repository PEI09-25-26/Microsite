# Usabilidade e acessibilidade digital

**March 2026 | 2nd Semester**

---

This section presents the usability and accessibility principles considered during the development of the Sueca application, as well as the evaluation methods used to assess the user experience.

---

# Design Principles

## Interface Consistency

The interface was designed to maintain a consistent visual identity across all screens.

- Screen layouts follow predictable patterns, making the application easier to learn.
- Navigation elements such as buttons, menus, and icons maintain the same structure and behaviour throughout the application.
- A consistent colour palette improves recognition and reduces confusion.

### Colour Usage

- **Orange buttons:** used for primary or more important actions.
- **Blue buttons:** used for secondary actions to maintain consistency.

## Navigation and Interaction

The application aims to provide simple and intuitive interaction flows.

- Only relevant information is presented to users, reducing cognitive load.
- Main tasks can be completed with a small number of steps.
- Interactive elements provide immediate visual feedback after user actions.
- Joining an online Sueca game can be completed in only **three steps**, improving efficiency.
