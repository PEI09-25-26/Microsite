# Accessibility Considerations

Accessibility principles were considered during the interface design process.

- High-contrast colours improve readability.
- Screens remain legible under different lighting conditions.
- Large buttons facilitate interaction on mobile devices.
- Clear and readable fonts enhance comprehension.
- A simple and structured layout reduces cognitive effort.
- Excessive animations were avoided to prevent distractions and improve usability.
