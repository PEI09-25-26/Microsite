# 10 Novembro

**Evolução do Módulo de Visão Artificial**

O módulo de Visão Artificial atingiu uma nova fase de desenvolvimento. A equipa focou-se na *otimização dos algoritmos de deteção de cartas*, resultando numa redução da taxa de erros.

**Iteração no Design do Protótipo Front-End**

Com base no feedback dos testes iniciais de usabilidade, o *protótipo do Front-End (no Figma)* sofreu uma *nova iteração de design*. Foram feitas melhorias na disposição dos elementos para otimizar a experiência do utilizador (UX) e garantir uma interface mais intuitiva e agradável, alinhada com as melhores práticas de design.

**Refatoração e Otimização do Protótipo**

A equipa de Back-End dedicou-se à refatoração do código do protótipo. O foco desta atividade foi aumentar a legibilidade, manter os padrões de código e otimizar o desempenho das rotinas críticas de processamento, preparando o código para a escalabilidade futura. Esta refatoração incluiu a reestruturação das classes principais de jogo e a limpeza de código redundante.
