# 27 Outubro

**Desenho e Suporte Físico**

Foram feitos *rabiscos e esboços preliminares do suporte físico para o telemóvel*. Este suporte é essencial para posicionar o dispositivo de forma estável e padronizada, garantindo a captação de imagem ideal para o módulo de Visão Artificial.

**Discussão de Ideias sobre o Suporte Físico**

Foi realizada uma *discussão de ideias aprofundada sobre o design e os requisitos do suporte físico* para o telemóvel. Os pontos chave abordados incluíram: a estabilidade do suporte, a altura ideal para a câmara (visão artificial), o material de construção (custo vs. durabilidade) e a facilidade de montagem e desmontagem. Esta discussão visa solidificar a decisão de design para o protótipo físico final.
