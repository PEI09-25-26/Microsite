# 17 Novembro

**Otimização Fina do Algoritmo de Deteção (Visão Artificial)**

A equipa de Visão Artificial dedicou-se à otimização dos parâmetros do algoritmo de deteção de cartas. Esta fase incluiu testes exaustivos com diferentes tipos de cartas, iluminações e ângulos de captura.

**Implementação de Logger e Monitorização no Back-End**

Foi implementado um sistema robusto de registo (Logger) no código *Back-End*. Este sistema é essencial para monitorizar o desempenho da API e rastrear erros durante as sessões de simulação e teste. A monitorização em tempo real das decisões do BOT está agora funcional, permitindo a análise detalhada do seu comportamento em diferentes cenários de jogo.
