# 15 Dezembro

**Estado Atual do Produto Mínimo Viável (MVP)**

Foi realizada uma *revisão abrangente do estado atual do MVP*. Concluiu-se que as funcionalidades críticas definidas para esta fase — incluindo a deteção básica de cartas, a simulação do motor de jogo (*Algoritmia*) e a interface de utilizador essencial (*Front-End*) — estão *quase totalmente implementadas e em fase de testes de integração inicial*. O MVP cumpre o objetivo de demonstrar a viabilidade técnica e a prova de conceito do sistema.

**Continuação do Desenvolvimento do MVP**

A equipa comprometeu-se a *continuar o desenvolvimento do MVP* para além das funcionalidades base. Os próximos *sprints* de desenvolvimento focar-se-ão na melhoria da performance dos subsistemas.. O objetivo é entregar um MVP mais robusto para os testes com utilizadores.

**Preparação para a Apresentação Oral da Milestone 3**

Foi iniciado o planeamento e a *preparação para a apresentação oral da Milestone 3*. O powerpoint foi feito pela Cátia e foram alinhadas as secções que cada membro irá apresentar.
